# OWASP_Web_Scanner
OWASP Web Scanner Project

목표 : owasp top 10 (2025) 중 자동화가 가능한 항목만 골라서 웹 취약점 스캐너 만든다.

## 1일차(2026.05.30) : 주어진 URL에 접속하고, 제목, 상태코드, 서버 정보 출력
완료한 작업:
- 코드에 지정된 URL을 파싱하였다.
- 서버 정보와 웹페이지 내용을 가져오는 기능을 구현하였다.
  
다음 단계:
- URL 입력 기능을 추가한다.
- 사용자가 입력한 URL의 웹페이지 내용을 가져올 수 있도록 구현한다.

현재 고민:
- 웹사이트마다 HTML 구조와 CSS 클래스명이 다르다.
- select_one()에 고정된 선택자를 사용하는 대신, 다양한 사이트에서도 동작할 수 있는 방법을 찾아야 한다

## 2일차(2026.05.31) : 웹페이지를 만들어 URL을 입력하고 해당 웹페이지에 있는 모든 링크 목록을 가져온다.
완료한 작업:
- Flask 웹페이지 생성, HTML 작성 -> URL 입력 기능 추가
- 접속한 URL 내의 모든 링크 파싱
- 불필요한 링크 제거

다음 단계 :
- 접속한 웹사이트 도메인만 남기고 다른 사이트 링크 제거
- 도메인을 절대경로로 출력
- 중복된 링크 제거
- ?id= 같은 파라미터 있는 URL 찾기

현재 고민 : 
- 오류 처리 ex)빈칸 입력, 잘못된 URL 입력
- 파라미터 찾을만한 사이트
